Diseño Web y Multimedia

Los componentes de una pagina web son: imagenes, texto, audio, video, animaciones y otros. Contenidos multimedia interactuando con usuarios de internet. Las secciones internas se ecuentran enlazadasde manera que el usuario puede navegar de una pagina a otra utilizando hipervinculos, un concepto de interactividad surgido con el fenomeno internet.
